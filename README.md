# Techy-Nerds
# Novara Capstone Project

A responsive React + Vite web application that visualizes planetary data in an interactive and user-friendly format. The project showcases information about planets in the solar system using API integration, responsive layouts, reusable React components, and modern frontend design principles.

# How To Run Locally

## 1. Clone the Repository

```bash
git clone https://github.com/M-rick-34/Techy-Nerds.git
```

## 2. Navigate Into the Project Folder

```bash
cd Novara-capstone-project/novara-capstone-project
```

## 3. Install Dependencies

```bash
npm install
```

## 4. Start the Development Server

```bash
npm run dev
```

## 5. Open in Browser

Visit:

```txt
http://localhost:5173
```
